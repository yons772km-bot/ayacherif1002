Présentation du cours

Ce cours présente les principales techniques utilisées pour séparer les constituants d’un mélange : la décantation, la filtration et la distillation.

Il permet de comprendre quelle méthode choisir selon le type de mélange : mélange hétérogène ou mélange homogène.

Objectif du cours : reconnaître les techniques de séparation, expliquer leur principe, identifier les appareils utilisés et distinguer le résidu, le filtrat et le distillat.

1. Introduction

Dans certains jus de fruits, on observe de la pulpe solide en suspension. Ces boissons sont des mélanges hétérogènes, car on distingue plusieurs constituants à l’œil nu.

Dans certains sirops, dans le café, le lait ou le vin, on ne distingue pas les constituants : ce sont des mélanges homogènes.

Problème : comment peut-on séparer les constituants d’un mélange homogène ou hétérogène ?

Dans ce cours, on étudie trois techniques de séparation :

  • la décantation ;
  • la filtration ;
  • la distillation.

2. La décantation

Définition : la décantation est une technique de séparation, sous l’effet de la gravitation, de plusieurs phases non miscibles dont l’une au moins est liquide ou gazeuse.

Elle permet de séparer des liquides non miscibles de densités différentes, ou des solides insolubles en suspension dans un liquide.

3. Décantation d’un mélange solide-liquide

Après agitation d’un mélange d’eau et de terre, les particules de terre se dispersent dans l’eau. Après un temps de repos, elles se déposent progressivement au fond du récipient sous l’effet de leur poids.

Mélange Observation Résultat
Eau + terre Les particules de terre tombent au fond après repos. Dépôt de terre + liquide plus clair.
Eau + sable Les grains de sable se déposent au fond. Séparation par décantation.
À retenir : dans un mélange solide-liquide hétérogène, les particules solides les plus lourdes se déposent au fond du récipient.

4. Décantation d’un mélange liquide-liquide

Pour séparer deux liquides non miscibles, comme l’eau et l’huile, on utilise une ampoule à décanter.

  1. On introduit le mélange dans l’ampoule à décanter.
  2. On laisse reposer le mélange.
  3. Les deux liquides se séparent en deux couches.
  4. Le liquide le plus dense se place en bas.
  5. On ouvre le robinet pour récupérer d’abord le liquide le plus dense.
Conclusion : les liquides non miscibles d’un mélange hétérogène peuvent être séparés par décantation. Le liquide le plus dense est récupéré en premier.

5. La filtration

L’eau boueuse décantée n’est pas toujours parfaitement claire. Elle peut encore contenir des particules solides en suspension. Pour la rendre limpide, on utilise la filtration.

Principe : la filtration consiste à faire passer un mélange hétérogène à travers un filtre qui retient les particules solides.

On verse le mélange sur un papier filtre placé dans un entonnoir. Les particules solides restent dans le filtre, tandis que le liquide traverse le filtre.

Élément Définition
Résidu Particules solides retenues par le filtre.
Filtrat Liquide homogène obtenu après passage à travers le filtre.
Filtre Matériau qui laisse passer le liquide mais retient les particules solides.
À retenir : lors d’une filtration, le résidu reste dans le filtre et le filtrat est recueilli dans le récipient.

6. La distillation

Définition : la distillation est un procédé de séparation des constituants d’un mélange homogène liquide, lorsque les températures d’ébullition des substances sont différentes.

Sous l’effet de la chaleur, les substances se vaporisent. La vapeur obtenue passe ensuite dans un réfrigérant où elle se liquéfie. Le liquide recueilli s’appelle le distillat.

Chauffage Vaporisation Refroidissement Condensation Distillat

7. Étapes de la distillation

Étape Explication
Chauffage Le mélange homogène est chauffé dans un ballon.
Vaporisation Le constituant le plus volatil se transforme en vapeur.
Refroidissement La vapeur passe dans un réfrigérant et se liquéfie.
Récupération Le liquide recueilli à la sortie du réfrigérant est le distillat.
Conclusion : la distillation permet de séparer certains constituants d’un mélange homogène. Elle repose sur la vaporisation suivie d’une condensation.
À retenir : le liquide recueilli par distillation s’appelle le distillat. Il peut être un corps pur.

8. Comparaison des techniques de séparation

Technique Type de mélange Principe Exemple
Décantation Hétérogène solide-liquide ou liquide-liquide Séparation par repos et gravitation Eau + terre ; eau + huile
Filtration Hétérogène solide-liquide Le filtre retient les solides Eau boueuse
Distillation Homogène liquide Vaporisation puis condensation Eau salée ou eau minérale

Résumé du cours

  • La séparation des constituants d’un mélange dépend de la nature du mélange.
  • La décantation permet de séparer les phases d’un mélange hétérogène par repos.
  • Dans une décantation liquide-liquide, le liquide le plus dense est récupéré en premier.
  • La filtration permet de séparer un solide insoluble d’un liquide.
  • Le résidu est la partie solide retenue par le filtre.
  • Le filtrat est le liquide obtenu après filtration.
  • La distillation permet de séparer certains constituants d’un mélange homogène.
  • La distillation repose sur la vaporisation puis la condensation.
  • Le liquide obtenu par distillation s’appelle le distillat.
  • Le distillat peut être un corps pur.

Exercices d’application

Exercice 1 : Compléter les phrases

Complète avec les mots suivants : filtration - décantation - filtrat - distillation.

  1. Lors de la ____________, les particules solides les plus lourdes tombent au fond du récipient.
  2. Lors de la ____________, les particules solides restent dans le filtre.
  3. Le liquide obtenu après filtration est appelé le ____________.
  4. La ____________ est utilisée pour passer d’un mélange homogène à un corps pur.

Exercice 2 : Le thé

Pour faire du thé, Sara ajoute des feuilles de thé dans de l’eau bouillante. Elle verse ensuite le mélange dans un filtre posé dans un entonnoir. Elle recueille enfin la boisson chaude dans une tasse placée sous l’entonnoir.

  1. Le mélange initial eau + feuilles de thé est-il homogène ou hétérogène ? Justifier.
  2. Quel est le rôle du filtre ?
  3. La boisson obtenue est-elle un corps pur, un mélange homogène ou un mélange hétérogène ?

Exercice 3 : Eau boueuse et salée

François possède un bocal contenant de l’eau boueuse et salée. Il veut séparer les constituants de ce mélange.

  1. Quelles sont les différentes expériences qu’il doit réaliser ?
  2. Quels appareils doit-il utiliser ?
  3. Quel résultat obtient-il après chaque expérience ?

Exercice 4 : Distillation

Une expérience permet de séparer un mélange homogène liquide par chauffage puis refroidissement de la vapeur.

  1. Comment s’appelle cette expérience ?
  2. Quel est le rôle du ballon ?
  3. Quel est le rôle du thermomètre ?
  4. Quel est le rôle du réfrigérant ?
  5. Comment s’appelle le liquide recueilli ?

Exercice 5 : Filtration

Un schéma présente une technique de séparation des constituants d’un mélange hétérogène.

  1. Comment se nomme cette technique ?
  2. Identifier le filtre, l’entonnoir, le mélange hétérogène et le filtrat.
  3. Où se trouve le mélange homogène obtenu ?
  4. Où se trouve le mélange hétérogène au début ?

Corrigé indicatif

Correction de l’exercice 1

  1. Lors de la décantation, les particules solides les plus lourdes tombent au fond du récipient.
  2. Lors de la filtration, les particules solides restent dans le filtre.
  3. Le liquide obtenu après filtration est appelé le filtrat.
  4. La distillation est utilisée pour passer d’un mélange homogène à un corps pur.

Correction de l’exercice 2

  1. Le mélange initial est hétérogène, car les feuilles de thé sont visibles dans l’eau.
  2. Le filtre retient les feuilles de thé et laisse passer le liquide.
  3. La boisson obtenue est un mélange homogène.

Correction de l’exercice 3

  1. Il doit d’abord réaliser une décantation ou une filtration pour éliminer la boue, puis une distillation pour séparer l’eau du sel dissous.
  2. Il utilise un filtre, un entonnoir, un récipient, puis un dispositif de distillation.
  3. Après filtration, il obtient un filtrat salé. Après distillation, il obtient de l’eau pure appelée distillat, et le sel reste dans le ballon.

Correction de l’exercice 4

  1. L’expérience s’appelle la distillation.
  2. Le ballon contient le mélange à chauffer.
  3. Le thermomètre permet de contrôler la température.
  4. Le réfrigérant refroidit la vapeur pour la liquéfier.
  5. Le liquide recueilli s’appelle le distillat.

Correction de l’exercice 5

  1. La technique est la filtration.
  2. Le filtre retient les particules solides, l’entonnoir supporte le filtre, le mélange hétérogène est versé au départ, et le filtrat est le liquide recueilli.
  3. Le mélange homogène obtenu se trouve dans le récipient sous l’entonnoir.
  4. Le mélange hétérogène se trouve au-dessus du filtre avant le passage.
