Présentation du cours

Ce cours présente la notion d’action mécanique. Il permet de comprendre comment un corps peut agir sur un autre corps, soit pour le mettre en mouvement, soit pour modifier son mouvement, le déformer ou le maintenir au repos.

Le cours explique aussi les différents types d’actions mécaniques : les actions de contact, les actions à distance, les actions localisées et les actions réparties.

Objectif du cours : définir une action mécanique, identifier l’acteur et le receveur, distinguer l’effet dynamique et l’effet statique, reconnaître les actions de contact et à distance, puis faire le bilan des actions exercées sur un corps.

1. Notion d’action mécanique

En physique, lorsqu’un objet agit sur un autre objet, on parle d’action mécanique.

Le corps qui exerce l’action est appelé acteur ou donneur. Le corps qui subit l’action est appelé receveur.

Définition : une action mécanique est l’effet d’un corps sur un autre corps. Elle peut mettre un corps en mouvement, modifier son mouvement, le déformer ou participer à son repos.

Illustration intégrée : action du pied sur le ballon.

2. Effets des actions mécaniques

Une action mécanique peut produire deux grands types d’effets : un effet dynamique ou un effet statique.

Type d’effet Signification Exemples
Effet dynamique Mettre un corps en mouvement ou modifier son mouvement : vitesse ou trajectoire. Le pied agit sur le ballon ; la raquette agit sur la balle.
Effet statique Mettre un corps au repos, le maintenir en équilibre ou le déformer. La table soutient un objet ; le fil soutient une lampe.
À retenir : l’effet dynamique concerne le mouvement. L’effet statique concerne le repos, l’équilibre ou la déformation.

Illustration intégrée : effet dynamique et effet statique.

3. Les types d’actions mécaniques

3.1 Action de contact

Une action est dite de contact lorsqu’il y a contact entre le corps qui exerce l’action et le corps qui subit l’action.

Type d’action de contact Définition Exemple
Action de contact localisée Le contact se fait sur une petite surface assimilée à un point. Le fil exerce une action sur une boule suspendue.
Action de contact répartie Le contact se fait sur une surface étendue ou en plusieurs points. La table exerce une action sur une boîte posée sur elle.

3.2 Action à distance

Une action est dite à distance lorsqu’il n’y a pas de contact direct entre l’acteur et le receveur.

  • La Terre exerce une action attractive sur les corps : c’est le poids.
  • Un aimant exerce une action magnétique sur certains objets métalliques.
  • Deux corps électrisés peuvent exercer une action électrique à distance.

Illustration intégrée : contact localisée, contact répartie et action à distance.

4. Bilan des actions mécaniques

Faire le bilan ou l’inventaire des actions mécaniques consiste à identifier le corps étudié, puis à chercher toutes les actions qui s’exercent sur ce corps.

Méthode : choisir le système étudié, identifier les corps qui agissent sur lui, puis préciser pour chaque action le donneur, le receveur, le type d’action et son effet.

Illustration intégrée : inventaire des actions exercées sur une boule en fer.

Exemple : système étudié : la boule en fer. Les actions exercées sur elle sont :

  • l’action de contact localisée exercée par le fil sur la boule ;
  • l’action à distance répartie exercée par l’aimant sur la boule ;
  • l’action à distance répartie exercée par la Terre sur la boule, appelée poids.

5. Modélisation des actions mécaniques

Une action mécanique peut être étudiée à partir de deux critères : son type et son effet.

Action mécanique Types Effets
Action de contact Localisée ou répartie Dynamique ou statique
Action à distance Toujours répartie Dynamique ou statique

Activités simples ajoutées

Activité 1 : action sur un ballon

Matériel : un ballon.

  1. Placer le ballon au repos sur le sol.
  2. Le pousser doucement avec le pied.
  3. Observer l’effet de l’action du pied sur le ballon.

Conclusion : le pied met le ballon en mouvement. L’action a un effet dynamique.

Activité 2 : déformation d’une éponge

Matériel : une éponge.

  1. Appuyer avec la main sur l’éponge.
  2. Observer la forme de l’éponge.
  3. Relâcher l’éponge et observer à nouveau.

Conclusion : la main déforme l’éponge. L’action a un effet statique.

Activité 3 : action d’un aimant

Matériel : un aimant et un petit clou en fer.

  1. Approcher l’aimant du clou sans le toucher.
  2. Observer le mouvement du clou.
  3. Identifier le type d’action.

Conclusion : l’aimant agit sur le clou sans contact. C’est une action mécanique à distance.

Résumé du cours

  • Une action mécanique est l’effet d’un corps sur un autre corps.
  • Le corps qui exerce l’action est l’acteur ou le donneur.
  • Le corps qui subit l’action est le receveur.
  • Une action mécanique peut mettre un corps en mouvement.
  • Elle peut modifier le mouvement d’un corps : trajectoire ou vitesse.
  • Elle peut déformer un corps ou le maintenir au repos.
  • L’effet dynamique concerne le mouvement.
  • L’effet statique concerne le repos, l’équilibre ou la déformation.
  • Une action de contact nécessite un contact entre l’acteur et le receveur.
  • Une action de contact peut être localisée ou répartie.
  • Une action à distance s’exerce sans contact direct.
  • Le poids est une action à distance exercée par la Terre sur un corps.
  • Faire le bilan des actions consiste à inventorier toutes les actions exercées sur le système étudié.

Exercices d’application

Exercice 1 : Compléter les phrases

Complète avec les mots suivants : action mécanique - de contact - à distance - localisée - répartie.

  1. Lorsqu’un système agit directement sur un autre système, on dit qu’il exerce une ____________.
  2. Une action ____________ est une action qui ne peut s’exercer sans contact entre l’acteur et le receveur.
  3. Une action ____________ est une action qui s’exerce sans contact entre l’acteur et le receveur.
  4. Une action ____________ s’exerce sur une partie du receveur suffisamment réduite pour être assimilée à un point.
  5. Une action ____________ s’exerce sur une zone trop grande pour être assimilée à un point.

Exercice 2 : Effet dynamique ou statique

Action de la branche sur la pomme
